# Swiggy Instamart - Hyperlocal Grocery Delivery Platform

A modern, full-stack grocery delivery platform built with React and Node.js, featuring instant delivery, AI-powered recommendations, and real-time tracking.

## 🚀 Features

- **Instant Delivery Promise**: Delivery in under 30 minutes with real-time tracking
- **Personalized Homepage**: ML-based recommendations and smart carousels
- **Smart Cart & Upselling**: AI suggestions and bundle recommendations
- **Scheduled Delivery**: Flexible time slots and recurring orders
- **Instamart+ Subscription**: Premium membership with exclusive benefits
- **Real-time Inventory**: Live stock updates and demand forecasting
- **Multi-language Support**: Hindi, Telugu, Tamil, Kannada support
- **Sustainability Mode**: Eco-friendly delivery options

## 🛠️ Tech Stack

- **Frontend**: React 18, TypeScript, Tailwind CSS, React Query
- **Backend**: Node.js, Express, TypeScript
- **Database**: PostgreSQL, MongoDB, Redis
- **Search**: Elasticsearch integration
- **AI/ML**: TensorFlow.js for recommendations
- **Real-time**: Socket.io for live tracking
- **Authentication**: JWT tokens
- **Payments**: Stripe, Razorpay integration

## 📦 Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd instamart
```

2. Install all dependencies:
```bash
npm run install-all
```

3. Set up environment variables:
```bash
cp server/.env.example server/.env
cp client/.env.example client/.env
```

4. Start the development servers:
```bash
npm run dev
```
